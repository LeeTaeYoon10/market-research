# 시장조사 도구 (로컬 실행)

제품과 소구점을 입력하면, 크롬으로 여러 사이트를 직접 검색해
관련 글의 **링크 + 요약 + 관련성 점수 + 발행 추이**를 정리해 줍니다. 검색 API 키 없이 동작합니다.

## 기능
- 멀티 소스 검색: 네이버(블로그·카페·뉴스·쇼핑), 다음/웹문서, **디시인사이드**
- **직접 링크 추가**: 원하는 글 URL을 넣으면 그 글도 함께 분석
- AI 요약·관련성 점수: 동음이의 노이즈(예: 그래니 살룬)를 걸러 정렬
- **조사 개요**: 어디서 어떤 검색어로 몇 건 분석했는지 표기
- **발행·언급 추이 그래프**: 수집글 발행 시점 분포(관심도 proxy)
- **심층 요약**(옵션): 상위 글 본문 전체를 읽어 깊은 요약
- 내보내기: 엑셀(CSV) / 마크다운 리포트 / 노션 페이지

## 처음 한 번만: 설치
`run.bat` 더블클릭 → 자동으로 환경을 설치하고 서버를 켭니다. (처음엔 몇 분 소요)

> Python 3.12 권장. (3.14는 일부 패키지 미지원) 설치된 게 없으면 `py install 3.12`.

## 사용
1. `run.bat` 실행 → 브라우저가 자동으로 http://127.0.0.1:5000 열림
2. 제품(예: 그래니 샐러드) + 소구점(예: 간편하지만 건강하게) 입력
3. 검색할 곳 선택 / (선택) 직접 링크 붙여넣기 / (선택) 심층 요약 체크
4. **조사 시작** → 조사 개요 · 추이 그래프 · 결과 표 확인
5. 엑셀·마크다운·노션으로 내보내기

## AI 요약 (키 불필요)
이 PC에 **클로드 코드가 설치·로그인**돼 있으면 자동으로 AI 요약이 켜집니다 (`claude -p` 사용).
없으면 미리보기 텍스트만 표시됩니다. (대안: 환경변수 `ANTHROPIC_API_KEY` 설정 시 API 사용)

## 노션 연동 (선택)
결과를 노션 페이지로 자동 생성하려면 폴더에 `config.py`를 만들고:
```python
NOTION_TOKEN = "secret_xxx"      # notion.so/my-integrations 에서 발급
NOTION_PARENT = "페이지ID32자리"   # 결과를 넣을 부모 페이지 ID
```
설정 방법: ① notion.so/my-integrations에서 Internal Integration 생성 → 토큰 복사
② 노션에서 결과를 담을 페이지 만들고 우측 ··· → 연결 → 만든 integration 추가
③ 그 페이지 URL 끝의 32자리가 NOTION_PARENT
(환경변수 `NOTION_TOKEN`·`NOTION_PARENT`로 넣어도 됩니다.)

## 회사 공유
이 폴더(market-research) 전체를 압축해 동료에게 전달 → 각자 PC에서 `run.bat` 실행.
AI 요약은 그 PC에 클로드 코드가 있어야 켜집니다(없으면 링크·미리보기만).

## 참고
- 트렌드 그래프는 '검색량'이 아니라 수집글의 발행 시점 분포입니다(키 없이 가능한 관심도 지표).
- 검색어는 '그래니 살라'보다 '그래니 샐러드'가 더 정확합니다(살라=동음이의 노이즈 많음).
